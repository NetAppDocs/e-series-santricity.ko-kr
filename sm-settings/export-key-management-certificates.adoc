= 키 관리 서버 인증서를 내보냅니다
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
키 관리 서버의 인증서를 로컬 컴퓨터에 저장할 수 있습니다.

.시작하기 전에
* 보안 관리자 권한이 포함된 사용자 프로필로 로그인해야 합니다. 그렇지 않으면 인증서 기능이 나타나지 않습니다.
* 인증서를 이전에 가져와야 합니다.


.단계
. 설정 [인증서] 메뉴를 선택합니다.
. 키 관리 * 탭을 선택합니다.
. 테이블에서 내보낼 인증서를 선택한 다음 * 내보내기 * 를 클릭합니다.
+
저장 대화 상자가 열립니다.

. 파일 이름을 입력하고 * 저장 * 을 클릭합니다.
