= 알림에 대한 이메일 주소를 추가합니다
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
최대 20명의 전자 메일 알림을 받는 사람을 추가할 수 있습니다.

.단계
. 설정 [알림] 메뉴를 선택합니다.
. 이메일 * 탭을 선택합니다.
. 이메일 추가 * 를 선택합니다.
+
이메일 추가 대화 상자가 열립니다.

. 빈 필드에 새 이메일 주소를 입력합니다. 둘 이상의 주소를 추가하려면 * 다른 이메일 추가 * 를 선택하여 다른 필드를 엽니다.
. 추가 * 를 클릭합니다.


.결과
경고 페이지의 이메일 탭에는 새 이메일 주소가 표시됩니다.
