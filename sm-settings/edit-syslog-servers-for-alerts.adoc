= 경고를 위해 syslog 서버를 편집합니다
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
syslog 알림을 수신하는 데 사용되는 서버 주소를 편집할 수 있습니다.

.단계
. 설정 [알림] 메뉴를 선택합니다.
. Syslog * 탭을 선택합니다.
. 테이블에서 syslog 서버 주소를 선택한 다음 오른쪽 끝에 있는 * Edit * (연필) 아이콘을 클릭합니다.
+
행이 편집 가능한 필드가 됩니다.

. 서버 주소와 UDP 포트 번호를 편집한 다음 * 저장 * (확인 표시) 아이콘을 클릭합니다.


.결과
업데이트된 서버 주소가 테이블에 나타납니다.
