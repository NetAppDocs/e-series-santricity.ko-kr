= 테이블에 사용할 수 없는 값이 표시되는 이유는 무엇입니까?
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
원격 스토리지 배열에 있는 데이터를 표시할 수 없는 경우 테이블에는 사용할 수 없는 값이 나열됩니다.

원격 스토리지 시스템 데이터를 표시하려면 Unified Manager에서 System Manager를 실행합니다.
