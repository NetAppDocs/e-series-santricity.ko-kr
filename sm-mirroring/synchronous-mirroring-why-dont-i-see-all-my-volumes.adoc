= 동기 미러링 - 내 볼륨을 모두 볼 수 없는 이유는 무엇입니까?
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
미러링된 쌍의 기본 볼륨을 선택하면 목록에 모든 적격 볼륨이 표시됩니다.

사용할 수 없는 볼륨은 해당 목록에 표시되지 않습니다. 볼륨은 다음과 같은 이유로 적합하지 않을 수 있습니다.

* 볼륨은 스냅샷 볼륨 또는 씬 볼륨과 같은 비표준 볼륨입니다.
* 볼륨이 최적이 아닙니다.
* 볼륨이 이미 미러링 관계에 있습니다.
