= 씬 볼륨을 모니터링합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
씬 볼륨에서 공간을 모니터링하고 적절한 경고를 생성하여 용량 부족 상태를 방지할 수 있습니다.

[NOTE]
====
EF600 또는 EF300 스토리지 시스템에서는 이 기능을 사용할 수 없습니다.

====
[NOTE]
====
SANtricity System Manager에서는 씬 볼륨을 생성하는 옵션을 제공하지 않습니다. 씬 볼륨을 생성하려면 CLI(Command Line Interface)를 사용합니다.

====
