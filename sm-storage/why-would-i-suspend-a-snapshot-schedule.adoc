= 스냅샷 스케줄을 일시 중지하는 이유는 무엇입니까?
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
스케줄이 일시 중지되면 예약된 스냅샷 이미지 생성이 수행되지 않습니다. 스토리지 공간을 절약하기 위해 스냅샷 스케줄을 일시 중지한 다음 나중에 예약된 스냅샷을 다시 시작할 수 있습니다.

스냅샷 스케줄이 필요하지 않은 경우 스케줄을 일시 중지하는 대신 삭제해야 합니다.
